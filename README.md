An attempt to understand it


# Legato Command line build

From project root  

**Set the cross build environment**  

source ~/legato/packages/legato.sdk.latest/resources/configlegatoenv

**Build for wp85**  

```
make wp85
```


** install from host (assumes target is 192.168.2.2)**  

```
make install 
```
